# 백업 파일 설명 (2025-11-06)

이 폴더는 프로젝트 정리 과정에서 백업된 레거시 파일들을 포함합니다.

## 폴더 구조
```
_backup_2025_11_06/
├── src/           # 이전 버전 소스 파일
│   ├── app.js    # 메인 애플리케이션 로직
│   ├── data.js   # 데이터 모델
│   ├── ui.js     # UI 유틸리티 함수
│   └── style.css # 스타일시트
├── prototypes/    # HTML 프로토타입 버전들
└── script.js     # 사용되지 않던 스크립트 파일
```

## 파일 설명

### src/ 폴더
- `app.js`: 이전 버전의 메인 애플리케이션 로직 (필터링, 상태 관리 등)
- `data.js`: 데이터 모델 및 더미 데이터
- `ui.js`: DOM 조작 유틸리티 함수들
- `style.css`: 기존 스타일시트

### prototypes/ 폴더
- `Support-matching-prototype-refactor.html`: 리팩토링 중간 버전
- `Spmc-prpe.html`: 초기 프로토타입 버전

### 기타
- `script.js`: 테스트용으로 추가되었던 빈 스크립트 파일

## 참고사항
- 이 파일들은 React 기반으로 재작성되기 전의 레거시 버전입니다.
- 코드 참조나 기능 비교가 필요할 때 사용할 수 있습니다.
- 실제 프로젝트에서는 `client/` 폴더의 React 버전을 사용해주세요.
