자주 묻는 질문 (FAQ)

1. 검색 결과가 없어요 — 어떻게 해야 하나요?
필터를 완화하거나 키워드를 단어 단위로 줄여보세요. 예: "전세자금 지원" 대신 "전세자금". 지역 필터를 "전국"으로 바꾸면 더 많은 결과를 확인할 수 있습니다.
2. 저장한 조건은 어디에 저장되나요?
브라우저의 localStorage에 저장됩니다. 같은 브라우저/기기에서만 복원 가능합니다. 서버 동기화 기능은 추후 제공 예정입니다.
3. 알림(Notifications)은 어떻게 동작하나요?
현재는 시연용으로 콘솔 로그를 통해 알림을 제공합니다. 실제 푸시/이메일 알림은 백엔드 연동이 필요하며 추후 추가될 예정입니다.
4. 관심태그를 클릭하면 무엇이 되나요?
카드의 태그를 클릭하면 해당 태그가 검색어로 적용되거나 관련 필터가 자동으로 설정되어 즉시 검색이 실행됩니다.
5. 자동완성에서 원하는 항목이 없어요.
검색창에 직접 입력하신 후 엔터를 눌러 검색하세요. 자동완성은 현재 로컬 태그 목록 기반으로 동작합니다.
6. 저장된 조건을 여러 개 만들 수 있나요?
네, 여러 조건을 이름을 붙여 저장할 수 있으며, 저장된 조건 목록에서 불러오기/삭제가 가능합니다.
7. 문의는 어떻게 하나요?
사이드바의 "문의하기"를 클릭하거나 문의 페이지에서 이메일/문의 폼으로 연락하실 수 있습니다.
8. 데이터 출처는 어디인가요?
현재 페이지는 데모용으로 샘플 데이터(mock)를 사용합니다. 실제 기관 데이터는 API 연동 시 제공될 예정입니다.
